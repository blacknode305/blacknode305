🎤 Озвучка текста

⚠ Браузерный SpeechSynthesis API не позволяет надежно сохранить системный голос как MP3 напрямую во всех браузерах. Эта версия сохраняет введённый текст в файл для дальнейшего использования в TTS-сервисах или офлайн-конвертерах. Для полноценного сохранения аудио потребуется сторонний TTS API (Google Cloud, ElevenLabs, Azure и др.).
